LTSDK登录模块文档


注意：SDK中用到了IDFA，在提审时需要勾选使用了此功能
           SDK中使用到了NSUserDefault存储，请在开发中不要清空此对象中的数据，避免发生问题

第一步,添加库

1，手动将所需的第三方库拖入到项目的Frameworks目录下

1，如果项目中没有 Frameworks（框架）组，请创建一个，方法是：在 Xcode 的项目导航器内右击项目，并选择 New Group（新建组）

2，手动将所需的文件拖入都Frameworks中（弹出框中需勾选:Copy items if needed）

登录核心包需添加LTGameSDKCore.framework
Sign in with Apple需添加LTPlatformApple.framework
游客登录需添加LTPlatformGuest.framework
Google登录需添加LTPlatformGoogle.framework、GoogleSignIn.bundle、GoogleSignIn.framework、GoogleSignInDependencies.framework
并在项目中添加系统库(添加方法)
LocalAuthentication.framework、SafariServices.framework、AuthenticationServices.framework、SystemConfiguration.framework
Facebook登录需添加LTPlatformFacebook.framework、FBSDKCoreKit.framework、FBSDKLoginKit.framework
并在项目中添加系统库(添加方法)
Accelerate.framework、libc++.tbd

2，cocopods安装三方库

集成下列必要库文件，在podfile文件中添加如下代码（如果没有podfile文件，则使用命令行工具进入到项目根目录执行 pod init ）

use_frameworks!
pod 'AFNetworking', '~> 3.2.1', :subspecs => ['Reachability', 'Serialization', 'Security', 'NSURLSession']

3，info.plist添加如下字段(使应用可以访问http开头的网络请求)

<key>NSAppTransportSecurity</key>
    <dict>
        <key>NSAllowsArbitraryLoads</key>
        <true/>
    </dict>

1,添加URL Schemes,集成某个第三方的登录就需要添加对应的值      Facebook值为fbxxxxxxxxx，Google值为com.googleusercontent.apps.xxxxxxxxx，微信值为xxxxxxxxx，QQ的值为tencentxxxxxxxxx。（xxxxxxxxx为从对应平台申请到的应用ID）

2,配置-ObjC
Build Setting -> Other Linker Flags 添加-ObjC

第二步，初始化LT应用

1，在APPdelegate.m文件中导入头文件

#import <LTGameSDKCore/LTLoginManager.h>

2,初始化应用,在AppDelegate.m的didFinishLaunchingWithOptions方法中添加如下代码
上线前请确认isDevelopmentModel的参数值为 NO

[[LTLoginManager sharedInstance] registLTAppID:@"从乐推平台申请的应用ID" appkey:@"从乐推平台申请的应用AppKey" isDevelopmentModel:YES];

3,在AppDelegate.m文件添加应用回调

-(BOOL)application:(UIApplication *)app
           openURL:(NSURL *)url
           options:(NSDictionary<UIApplicationOpenURLOptionsKey,id> *)options{
    [[LTLoginManager sharedInstance] LTPlatformApplication:app openURL:url ptions:options];
    return YES;
}

一，开始使用带UI的SDK登录功能

1>在要使用UI登录功能的UIViewController中导入头文件

#import <SDKUISHYU/LTUIManager.h>
#import <LTGameSDKCore/LTUser.h>

2>设置用户协议和隐私条款连接地址（此处为演示地址，上线前请修改）

[[LTUIManager sharedInstance] linkOfUserAgreement:@"https://github.com/"
                                   andPrivacyLine:@"https://www.baidu.com/"];

3>调用UI界面进行显示

    [[LTUIManager sharedInstance] showLoginUI:self withBlock:^(LTUser * _Nonnull loginUser) {
        if (loginUser.stateCode == LTLoginSuccess) {
            NSString *lt_UID = loginUser.LT_UID;
            NSString *LTUidToken = loginUser.LTUidToken;
            NSLog(@"LT_UID is %@ ::: LT_UID_Token is %@",lt_UID,LTUidToken);
        }else if (loginUser.stateCode == LTLoginCancel){
            NSLog(@"用户取消了登录");
        }else if (loginUser.stateCode == LTLoginFailed){
            NSLog(@"登录失败");
        }
        [self checkUserInfo:loginUser];
    }];

二,附属操作

1>带有登录界面的退出登录操作

3,获取用户信息

    LTUser *current = [[LTLoginManager sharedInstance] getUserInfo];
    if (current.stateCode == LTLoginSuccess) {
        NSString *lt_UID = current.LT_UID;
        NSString *LTUidToken = current.LTUidToken;

    }

4,自动登录调用（根据缓存的第三方数据请求服务器）

    [[LTLoginManager sharedInstance] getUserStateWithServerCheck:^(LTUser * _Nonnull loginUser) {
        if (loginUser.stateCode == LTLoginSuccess) {
            NSString *lt_UID = loginUser.LT_UID;
            NSString *LTUidToken = loginUser.LTUidToken;
            NSLog(@"LT_UID is %@ ::: LT_UID_Token is %@",lt_UID,LTUidToken);
        } else {
            NSLog(@"User not login.");
        }
    }];

第五步，手机登录相关

在要使用手机登录相关功能的.m文件中导入头文件

#import <LTPlatformPhone/LTPhoneManager.h>

1,发送验证码

    //获取输入的手机号
    NSString *phoneNumber = self.phoneInput.text;
    [[LTPhoneManager sharedInstance] sendPhoneCode:^(NSDictionary * _Nonnull resultInfo) {
        NSString *authCode = resultInfo[@"data"][@"auth_code"];
        NSString *msg = resultInfo[@"msg"];
    } phoneNumber:phoneNumber];

2,手机号注册

    //获取输入的手机号
    NSString *phoneNumber = self.phoneInput.text;
    //获取输入验证码
    NSString *authCode = self.authCodeInput.text;
    //获取输入密码
    NSString *passsword = self.passwordInput.text;
    [[LTPhoneManager sharedInstance] phoneRegist:^(LTUser * _Nonnull loginUser) {
        if (loginUser.stateCode == LTLoginSuccess) {
            NSString *lt_UID = loginUser.LT_UID;
            NSString *LTUidToken = loginUser.LTUidToken;
            NSLog(@"LT_UID is %@ ::: LT_UID_Token is %@",lt_UID,LTUidToken);
        }
    } withPhoneNumber:phoneNumber withAuthCode:authCode withPassword:passsword];

3,手机号登录

    //获取输入的手机号
    NSString *phoneNumber = self.phoneInput.text;
    //获取输入的密码
    NSString *passsword = self.passwordInput.text;
    [[LTPhoneManager sharedInstance] phoneLogin:^(LTUser * _Nonnull loginUser) {
        if (loginUser.stateCode == LTLoginSuccess) {
            NSString *lt_UID = loginUser.LT_UID;
            NSString *LTUidToken = loginUser.LTUidToken;
            NSLog(@"LT_UID is %@ ::: LT_UID_Token is %@",lt_UID,LTUidToken);
        }
    } withPhoneNumber:phoneNumber withPassword:passsword];

4,重置密码


    //获取输入的手机号
    NSString *phoneNumber = self.phoneInput.text;
    //获取输入的验证码
    NSString *authCode = self.authCodeInput.text;
    //获取输入的密码
    NSString *passsword = self.passwordInput.text;
    [[LTPhoneManager sharedInstance] resetPassword:^(NSDictionary * _Nonnull resultInfo) {
        NSString *msg = resultInfo[@"msg"];
    } withPhoneNumber:phoneNumber withAuthCode:authCode withPassword:passsword];
